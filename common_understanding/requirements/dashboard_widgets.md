# Dashboard Widgets

## MVPウィジェット

## 1. Todo

### 目的

今日やるべきこと、大タスク、サブタスク、小タスクを確認して実行へ移る。

### 表示内容

- 大タスク
- 優先度
- カテゴリ
- 完了状況
- 数値が明確な小タスク
- 詳細展開でサブタスク

## 2. 学習度合い

### 目的

日々の頑張りを可視化し、学習継続の手応えを得る。

### 表示内容

- 今日の学習時間
- 今週の学習時間
- 完了タスク数
- 継続日数
- 応用情報の進捗
- AI評価またはAIコメント

## 3. ショートカット

### 目的

学習・開発・情報収集へすぐ移れる起動台にする。

### 表示内容

- 教材サイト
- ニュース
- TED
- 開発プロジェクト
- カフェ・自然スポット
- その他よく使うWebサイト

## 4. ログ・日記入力

### 目的

学習ログや日記を短い操作で残す。

### 表示内容

- 学習ログ追加
- 日記入力
- ジャンル選択
- タグ選択
- メモ
- 実績時間

## 5. AI秘書パネル

### 目的

現在の状況を短く整理し、次の行動に移りやすくする。

### 表示内容

- 今日の要約
- 次にやる候補
- 頑張りの短評
- 学習ログの整理
- 未完了Todoの軽いリマインド

### トーン

厳しすぎず、秘書・分析官として客観的に整理する。
