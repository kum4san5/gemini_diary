# Common Understanding

このディレクトリは、AI English Diary を「人生のダッシュボード」へ育てるための共通認識を置く場所です。

## 目的

- ユーザーとCodexが合意した仕様、方針、制約、優先順位を記録する
- 口頭・チャット上の曖昧なアイデアを、実装可能な単位へ変換する
- 後から見返したときに「なぜその設計にしたか」が分かる状態にする

## ファイル

- `product_vision.md`: プロダクトの目的、利用者像、成功状態
- `hearing_questions.md`: ヒアリング項目
- `open_questions.md`: 未決定事項
- `decisions.md`: 決定済み事項
- `feature_candidates.md`: 今後の機能候補
