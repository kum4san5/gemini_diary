# Hearing Questions

この質問集は、アプリを「人生のダッシュボード」として設計するためのヒアリング項目です。

## 1. 北極星

1. このアプリが1年後に最高の形で育ったとして、あなたは毎日どんな場面で開きますか？
2. 「人生のダッシュボード」と聞いたとき、あなたにとって絶対に外せない体験は何ですか？
3. このアプリがあることで、あなたの人生の何が変わっていてほしいですか？
4. 逆に、このアプリが絶対になってはいけないものは何ですか？
5. 現在の英語日記機能は、最終形の中で主役ですか、入口ですか、一機能ですか？

## 2. 対象領域

1. 記録したい領域を優先順に並べるとどうなりますか？

   - 仕事・開発
   - 健康
   - 睡眠
   - 運動
   - 食事
   - 気分・感情
   - 人間関係
   - お金
   - 読書・学習
   - 習慣
   - 目標・プロジェクト
   - アイデア
   - 旅行・イベント
2. 最初のMVPで扱うなら、どの3領域に絞りたいですか？
3. 「これは今は入れなくてよい」という領域はありますか？
4. 記録対象は定量データ中心ですか、文章・内省中心ですか？

## 3. 日々の入力体験

4. 逆に、毎日聞かれると面倒な質問は何ですか？
5. 入力形式はどれがよいですか？
   - 自由記述
   - チェックボックス
   - スライダー
   - 数値入力

   - チャット形式
   - テンプレート選択
6. 英語日記は日本語メモから自動生成したいですか、それとも英語で書く訓練を残したいですか？
7. 入力忘れに対する扱いは、厳しめ・優しめ・完全放置のどれがよいですか？

## 4. AIの役割

1. AIには先生、秘書、コーチ、分析官、友人のどれに近い振る舞いをしてほしいですか？
2. AIから毎日ほしいアウトプットは何ですか？

   - 感情分析
   - 明日の提案
   - 週次レビュー
   - 褒め・励まし
   - 厳しめの指摘

4. AIに踏み込んでほしくない領域はありますか？
5. AIの提案は「行動に落とす」ことを重視しますか、「気づき」を重視しますか？
